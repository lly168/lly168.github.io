ES6之变量的解构赋值

2019-05-15

解构赋值

以前为变量赋值，只能直接指定值

let a=100;
let b=200;
let c=300;

ES6允许写成下面这样

let [a,b,c]=[100,200,300]

上面代码表示，可以从数组中提取值，按照对应位置，对变量赋值。

  • 注意 :左右两边，结构格式要保持一致

let json= {
name:’leying’
age:18
job:码农;
};
let { name,age, job }=json
console.log(name,age,job) //leying leying 码农

let [ a,b,c ]=[ 1,2,3 ]
console.log(a,b,c) //1,2,undefined
如果解构不成功，变量的值就等于undefined。

解构赋值允许指定默认值。

let [ a,b,c=0 ]=[ 1,2,3 ]
console.log(a,b,c) //1,2,0
解构赋值允许指定默认值。

  • 上面代码中，如果一个数组成员是null，默认值就不会生效，因为null不严格等于undefined

解构赋值实例交换两个数

let a=10
let b=200
[ a,b ]=[ b,a ]
console.log(a,b) //200 10
